# 🛡️ EchoProof — AI Fake Review Detector

EchoProof is an AI-powered fake review detection system designed to identify suspicious, spam-like, and AI-generated product reviews using Natural Language Processing (NLP).

## 🚀 Features

- Fake review probability analysis
- Review authenticity score
- Sentiment analysis
- Suspicious keyword detection
- Clean interactive UI using Streamlit

## 🛠️ Tech Stack

- Python
- Streamlit
- TextBlob
- NLP
- Scikit-learn

## 📌 How It Works

The system analyzes:
- exaggerated promotional language
- spam-like phrases
- suspicious review patterns
- sentiment polarity

and generates an authenticity score for the review.

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
